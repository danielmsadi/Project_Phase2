# LIU University React Website

## Project Description
This project is a frontend web application for Lebanese International University. It was developed using ReactJS as Phase 2 of the Web Programming course project.

The website helps students learn about the university, view programs, register for a major, check admission requirements, and contact the university.

## Features
- Home page with university introduction
- About page with university information
- Programs page with program registration
- Ability to drop a registered program
- Admission page with required documents checklist
- FAQ accordion in the Admission page
- Contact page with form validation
- Responsive design for desktop and mobile

## Technologies Used
- ReactJS
- JavaScript
- HTML
- CSS
- React Router
- Git and GitHub
